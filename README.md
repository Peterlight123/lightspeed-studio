# Run and Deploy Your AI Studio App

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/e5cc7073-95d7-4f93-895a-5da4eeb3b1fc

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:  
   `npm install`
2. (Optional) If you have an API key in the future, you can set it in `.env.local`  
   `API_KEY=your_api_key_here`
3. Run the app:  
   `npm run dev`
